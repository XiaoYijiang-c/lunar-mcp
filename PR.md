## lunar-mcp

Chinese Calendar MCP Server - 中国农历 MCP 服务器

**28+ 工具函数**，支持：
- 农历日期查询
- 八字计算与分析
- 节气、节日查询
- 吉日/择日
- 合婚、起名
- 易经占卜、九星飞宫

**技术栈**：Go + MCP (JSON-RPC 2.0) + Docker

```bash
# Docker 运行
docker run -d -p 8080:8080 ghcr.io/xiaoyijiang-c/lunar-mcp:latest
```
